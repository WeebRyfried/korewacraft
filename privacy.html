Privacy Policy for mc.korewacraft.xyz

By continuing to use this Minecraft server, certain information is collected and retained, which includes, but is not limited to...

a. any IP address used to connect to the server
b. your Minecraft Username and/or UUID
c. any messages sent using the in-game chat function.

DISCLAIMER: We use a plugin called Dynmap which makes messages publicly available from the web in real time. (currently accessible from http://mc.korewacraft.xyz:8123)

Last updated 06/02/2020 at 11:33PM EST

NOTE: There's not really any easy way to disable logging. The data collected is a function of how Minecraft servers and how many other game servers work. We only plan to use logs for diagnostics (i.e. server crashes, conflicting plugins, etc.)
